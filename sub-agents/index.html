💡 Day 1 Pro Tip

Focus on understanding the "why" before the "how". Sub-agents solve the context pollution problem that occurs when one conversation handles too many different tasks. Think of them as specialized team members, each with their own expertise.

Start with Claude-generated agents and then customize them. Use phrases like "use PROACTIVELY" in descriptions to encourage automatic delegation. Remember: single responsibility principle applies to sub-agents too!
